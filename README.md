# Metroplex Pros - Dallas Home Improvement Services

A modern, professional website for Metroplex Pros home improvement business located in Dallas, Texas.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern Next.js application
- Responsive design
- Professional home improvement business showcase
- Contact forms and business information
- Project gallery
- Service listings

## Pages

- Home (/) - Business overview and hero section
- About (/about) - Company information and team
- Services (/services) - Home improvement services offered
- Gallery (/gallery) - Showcase of completed projects
- Contact (/contact) - Contact information and forms
